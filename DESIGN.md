# 公务员结构化面试训练小程序 - 设计风格指南

## 产品定位

**应用类型**：公务员面试备考训练工具  
**目标用户**：准备公务员结构化面试的考生  
**核心场景**：模拟真实面试环境，AI智能评分，专业点评反馈，帮助考生系统提升面试能力

## 气质与意象

**关键词**：专业、沉稳、清晰、高效

**场景意象**：一间安静的模拟面试教室，光线明亮但不刺眼，桌上整齐摆放着评分表和答题卡，墙上挂着简洁的计时器，空气中弥漫着专注备考的氛围。考官（AI）以温和专业的语调给出点评，考生在反复训练中逐渐找到自信。

## 视觉策略

- **整体风格**：政务风 + 现代简约，避免花哨装饰
- **图形语言**：圆角卡片、清晰图标、数据可视化图表
- **信息层级**：突出核心操作（答题、查看成绩），次要信息用淡色区分

## 配色方案

| 用途 | Tailwind 类名 | 说明 |
| --- | --- | --- |
| 主色（品牌色） | `bg-blue-600` / `text-blue-600` | 稳重、专业，政务风格 |
| 辅色 | `bg-blue-50` / `text-blue-500` | 用于卡片背景、次要按钮 |
| 成功色 | `bg-green-500` / `text-green-600` | 高分提示、进步反馈 |
| 警告色 | `bg-orange-500` / `text-orange-600` | 中等分数、改进提醒 |
| 错误色 | `bg-red-500` / `text-red-600` | 低分提示、错误信息 |
| 中性背景 | `bg-gray-50` / `bg-white` | 页面背景、卡片 |
| 文字色 | `text-gray-900` / `text-gray-600` | 主文字、次要文字 |
| 边框色 | `border-gray-200` | 分隔线、卡片边框 |

## 字体排版

- **主字体**：系统默认（PingFang SC / 苹方），中文友好
- **标题**：`text-xl font-semibold` (20px, 半粗)
- **副标题**：`text-lg font-medium` (18px, 中等)
- **正文**：`text-base` (16px)
- **辅助文字**：`text-sm text-gray-500` (14px, 淡色)
- **评分数字**：`text-2xl font-bold text-blue-600` (突出显示)

## 间距系统

- **页面边距**：`p-4` (16px)
- **卡片内边距**：`p-4` 或 `p-6`
- **组件间距**：`gap-4` (16px) 用于列表项
- **分组间距**：`mb-6` (24px) 用于模块分隔

## 组件使用原则

- **按钮**：优先使用 `@/components/ui/button`，主操作用 `variant="default"`，次要用 `variant="outline"`
- **卡片**：优先使用 `@/components/ui/card` 展示题目、评分、报告
- **输入框**：优先使用 `@/components/ui/input` 和 `@/components/ui/textarea`（答题输入）
- **标签**：优先使用 `@/components/ui/badge` 展示题型标签、评分档位
- **进度条**：优先使用 `@/components/ui/progress` 展示计时进度
- **选择器**：优先使用 `@/components/ui/tabs` 展示题型切换
- **弹窗**：优先使用 `@/components/ui/dialog` 展示评分详情、点评内容
- **Toast**：优先使用 `@/components/ui/toast` 提示操作结果
- **表格**：优先使用 `@/components/ui/table` 展示训练报告数据

## 导航结构

**TabBar 配置**（4个主要入口）：

| 页面 | 图标 | 说明 |
| --- | --- | --- |
| 首页 | House | 快速开始训练、今日推荐题目 |
| 题库 | FileText | 题目分类浏览、筛选 |
| 记录 | History | 答题记录、错题收藏 |
| 我的 | User | 个人成绩、训练报告、设置 |

**页面跳转规范**：
- TabBar 页面使用 `Taro.switchTab()`
- 详情页使用 `Taro.navigateTo()`

## 功能页面规划

### 1. 首页
- 快速入口：单题练习、全真模拟
- 今日推荐题目卡片
- 最近训练进度概览

### 2. 题库列表页
- 题型筛选 Tabs（综合分析、组织协调、人际沟通、应急应变）—— **四大题型**
- 六维度评分（综合分析能力、组织协调能力、人际沟通能力、应急应变能力、语言表达、举止仪表）
- 题目列表卡片（标题、类型标签、难度标记）
- 搜索功能

### 3. 答题页
- 题目展示区（标题、要求）
- 计时器（审题3分钟 + 答题3分钟）
- 答案输入区（Textarea）
- 录音按钮（可选语音答题）
- 提交按钮

### 4. 评分结果页
- 六维度评分展示（雷达图/条形图）
- 分档评级（好/中/差）
- 总分展示
- 查看点评按钮

### 5. 专业点评页
- 优点亮点
- 不足之处
- 改进建议
- 高分答题思路参考

### 6. 训练报告页
- 得分趋势折线图
- 题型正确率统计
- 薄弱环节分析
- 练习次数统计

### 7. 错题收藏页
- 错题列表（低分题目）
- 好题收藏（高分题目）
- 重新练习入口

## 小程序约束

- **包体积**：主包 ≤ 2MB，总包 ≤ 20MB
- **图片策略**：所有图片使用 TOS 对象存储，代码中用 URL 引用
- **性能优化**：虚拟列表加载题目，避免一次性渲染大量数据
- **录音权限**：需在小程序后台配置录音权限说明

## 设计禁忌

- ❌ 禁止使用花哨渐变、卡通插画（不符合政务严肃风格）
- ❌ 禁止使用过多装饰性图标（保持简洁专业）
- ❌ 禁止用 `View/Text` 手搓按钮、卡片、标签等通用组件
- ❌ 禁止硬编码 px 尺寸（使用 Tailwind 预设类名）
- ❌ 禁止将图片打包到项目中（使用 TOS 存储）
