# 公众号关键词设置指南

## 一、登录微信公众平台

1. 打开浏览器访问：https://mp.weixin.qq.com
2. 使用公众号管理员微信扫码登录
3. 确保登录的是「上岸公考面试」公众号

---

## 二、设置关键词自动回复

### 步骤：

1. 进入「自动回复」菜单（左侧导航栏）
2. 点击「关键词回复」
3. 点击「添加规则」

### 规则配置：

| 配置项 | 内容 |
|--------|------|
| **规则名称** | 会员开通 |
| **关键词** | 会员（勾选「全匹配」） |
| **回复内容** | 见下方模板 |

### 回复内容模板：

```
🎉 欢迎加入「上岸吧公考面试」！

您已进入会员开通通道。

📱 开通步骤：
1. 点击下方链接打开小程序
2. 进入「会员」页面
3. 添加客服微信：gxmasheng
4. 发送「开通会员 + 套餐类型」
5. 客服为您开通后，刷新小程序即可使用

💎 套餐类型：
• 月卡 29.9元：30天全功能 + 1次人工点评
• 季卡 59.9元：90天全功能 + 2次人工点评
• 年卡 199.9元：365天全功能 + 6次人工点评

如有疑问，请直接回复「客服」获取帮助。
```

---

## 三、设置公众号菜单

### 步骤：

1. 进入「自定义菜单」（左侧导航栏）
2. 添加菜单按钮：

| 菜单名称 | 跳转类型 | 目标 |
|----------|----------|------|
| 题库练习 | 小程序 | 上岸吧公考面试小程序题库页 |
| 开通会员 | 小程序 | 上岸吧公考面试小程序会员页 |
| 客服咨询 | 跳转网页 |客服微信二维码页面 |

3. 点击「保存并发布」

---

## 四、其他建议设置

### 1. 被关注自动回复

进入「自动回复」→「被关注回复」，设置：

```
欢迎关注「上岸吧公考面试」！

🎯 我们提供：
• AI智能评分系统
• 全真模拟面试训练
• 专业老师人工点评
• 备考指南与素材库

📱 点击底部菜单「题库练习」开始训练

回复「会员」获取会员开通指南
回复「客服」获取客服联系方式
```

### 2. 消息自动回复

进入「自动回复」→「消息自动回复」，设置：

```
感谢您的留言！

如需会员开通，请回复「会员」
如需客服帮助，请回复「客服」

我们会在24小时内回复您。
```

## 五、验证设置

完成设置后，请测试：

1. 用另一个微信号关注公众号
2. 发送「会员」关键词
3. 确认收到自动回复
4. 点击菜单按钮测试跳转
