## 二、客服后台开通会员操作

### 操作步骤
1. 用户添加客服微信后发送「会员开通 +套餐类型」
2. 用户付款后，客服登录后台管理系统
3. 进入「用户管理」页面
4. 输入用户ID（用户提供）
5. 选择套餐类型，点击「开通会员」
6. 用户刷新小程序，会员状态生效

### 后台管理入口
- 地址：小程序首页 → 我的 → 后台管理
- 密码：admin123（可在代码中修改）
